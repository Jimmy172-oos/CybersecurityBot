# Cybersecurity Awareness Bot

A console-based chatbot built in C# that educates South African citizens about cybersecurity threats.

## How to Run
1. Clone this repository
2. Open `CybersecurityBot.sln` in Visual Studio 2022
3. Ensure `greeting.wav.wav` is in the project folder with "Copy if newer" set
4. Press F5 to run or run from the top of the screen a lime button with 'Cybersecuritybot' next to it.

## Features
- ASCII art logo
- Voice greeting on startup after the logo has appeared
- Personalised user interaction
- Responses on: phishing, passwords, safe browsing, malware
- Input validation
- Typing effect for a conversational feel
